Description

The task for this project was to design and build a two-player game utilizing object oriented programming. I grew up playing all sorts of card games, one of my favorites being speed. So I decided to model it. Some of the more difficult logic problems required out of the box thinking, such as using the JQuery .eq() method to implement a looping select feature allowing a card to be chosen from the player's hand. The app was built with HTML, CSS, Javascript, and JQuery.
