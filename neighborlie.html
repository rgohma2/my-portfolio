Description

This app was built for the 2021 Tandom Software Engineering Internship coding challenge. To be considered for the application, it was required to create a triva game from JSON data consisting of an array of questions that are each objects with the multiple choice answers, the incorrect answers, and the correct answer. I built the framework of the app using Semantic UI and met the requirements of the challenge while adding some unique features of my own: a progress bar that updates in real time after the user submits an answer, an option to return to the main menu during a round, as well as two modes of difficulty with different sets of questions.

I had only found out about the application for the internship the morning of the day that it was to be submitted for consideration, so it was somewhat of a speed coding challenge. I opted to use React and Semantic UI to build it because I am very comfortable in both and I knew I could accomplish what I set out to do in a short amount of time. If I had more time I would have injecting a little more of my own creativity my means of CSS styling, although I feel the layout and functionality do their due dilligence in this regard.
